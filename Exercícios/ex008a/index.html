Principais formatações

Negrito/destaque

Nesse parágrafo fazemos o uso da tag <b> para deixar esse texto em negrito.

Nesse parágrafo fazemos o uso da tag <strong> para deixar esse texto em negrito

itálico/ênfase

Nesse parágrafo fazemos o uso da tag <i> para deixar esse texto em itálico.

Nesse Parágrafo fazemos o uso da tag <em> para deixar esse texto em itálico.

Marca texto

Nesse Parágrafo fazemos o uso da tag <mark> para deixar esse texto marcado.

texto grande e texto pequeno

Nesse Parágrafo fazemos o uso da tag <big> e <small> para deixar esse texto grande e esse texto pequeno.

Texto riscado

Podemos usar a tag <del> para indicar que o texto em questão não deve ser considerado, dessa forma.

Texto inserido

Podemos usar a tag <ins> para indicar que ele foi adicionado depois, dessa forma.

Podemos usar também a tag <u> para texto sublinhado (mesma função, porém não semântica)

Texto sobrescrito

no texto sobrescrito podemos elevar caracteres por exemplo: x2+40

Texto subscrito

No texto subscrito podemos colocar caracteres abaixo de outros, por exemplo: h2o
